Setup Guide

TV Connectivity Hub
עברית

1) Install

  1. Download and run installer from homepage.
  2. Use main channel unless you need release.
  3. Verify versionName/versionCode in installer output.

2) Enable access

  1. Accessibility: ON
  2. Notification Access: recommended
  3. Battery optimization exemption: recommended
  4. Write Settings: optional but recommended

3) Start session

Press Start Utility and verify readiness checks become YES.

4) Troubleshooting

  • If Notification Access = NO, open notification listener settings.
  • If Battery = NO, exempt app from optimization.
  • If Write Settings = NO, grant permission from Power menu.
