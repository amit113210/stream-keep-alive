Step 1 - Install

  1. Run the Mac/Windows installer.
  2. Select main channel (recommended for most users).
  3. Confirm installed app version in installer output (`versionName/versionCode`).

Step 2 - Required access

  1. Accessibility: required.
  2. Notification Access: strongly recommended.
  3. Battery optimization exemption: recommended for reliability.
  4. Write Settings: optional but recommended for timeout hardening.

Step 3 - Readiness check

Open the app and verify checklist rows turn to YES. If something is missing, use Fix buttons or Power & System helper.

Common troubleshooting

  • Notification Access = NO → open notification listener settings and grant access.
  • Battery Exemption = NO → exempt the app from battery optimization.
  • Write Settings = NO → grant permission via Power & System menu.
  • Timeout Override = NO → grant Write Settings and restart utility session.
