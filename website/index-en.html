Core features

1) Connectivity & hotspot helper

Quick entry points for Wi-Fi/network/hotspot flows with OEM-specific fallbacks.

2) Power & System helper

Central menu for Battery Optimization, Write Settings, App Info, Accessibility, and Notification Access.

3) Readiness dashboard

YES/NO visibility for Accessibility, Notification Access, Battery Exemption, and Write Settings.

4) Troubleshooting telemetry

Runtime diagnostics for advanced users: service state, permissions, timeout hardening, and more.

Quick setup

  1. Install using the Mac/Windows installer (channel main recommended).
  2. Enable Accessibility service for the app.
  3. Enable Notification Access (recommended).
  4. Open Power & System helper and complete missing permissions.
  5. Verify checklist turns to YES states.

Permissions explained

Accessibility
Required for local utility actions and readiness checks.
Notification Access
Improves runtime signal visibility and diagnostics quality.
Battery Exemption
Recommended for continuity on aggressive OEM power management.
Write Settings
Optional but recommended for display-timeout hardening.

Known limitations

Android TV behavior differs by OEM and firmware version. Some shortcut intents are device-specific. The app provides best-effort fallbacks but cannot guarantee identical behavior on every model.
