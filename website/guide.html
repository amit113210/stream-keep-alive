Setup Guide

TV Connectivity Hub for Android TV

1) Install

  1. Download and run installer from homepage.
  2. Use channel main unless you explicitly need release channel.
  3. Confirm installed versionName/versionCode at the end of installer output.

2) Enable required access

  1. Accessibility service: ON
  2. Notification Access: Recommended ON
  3. Battery optimization exemption: Recommended
  4. Write settings permission: Optional but recommended for display timeout hardening

3) Start Utility Session

Open TV Connectivity Hub and press Start Utility. Verify checklist items become YES.

4) Troubleshooting checklist

  • If Notification Access is NO: open notification listener settings.
  • If Battery Optimization is NO: open battery settings and exempt the app.
  • If Write Settings is NO: grant permission from the power helper menu.
  • If timeout override is not active: re-check Write Settings and restart session.

Important limitations

Android TV behavior varies by OEM firmware. This app provides best-effort utility hardening and diagnostics and does not guarantee identical behavior across all apps/devices.
