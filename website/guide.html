Setup Guide | מדריך התקנה

TV Connectivity Hub for Android TV | מרכז קישוריות לטלוויזיה

1) Install | התקנה

  1. Download and run installer from homepage.
    הורד והרץ מתקין מדף הבית.
  2. Use channel main unless you need release specifically.
    השתמש בערוץ main אלא אם צריך release ספציפית.
  3. Verify installed versionName/versionCode in installer output.
    אמת גרסה מותקנת בפלט המתקין.

2) Enable access | הפעלת גישות

  1. Accessibility service: ON | נגישות: פעיל
  2. Notification Access: Recommended ON | גישת התראות: מומלץ להפעיל
  3. Battery optimization exemption: Recommended | החרגת סוללה: מומלץ
  4. Write settings: Optional but recommended | Write settings: אופציונלי אך מומלץ

3) Start utility session | התחלת סשן

Press Start Utility and verify checklist items become YES.
לחץ Start Utility ובדוק שהצ'קליסט עובר ל-YES.

4) Troubleshooting | פתרון תקלות

  • Notification Access = NO: open notification listener settings.
    אם Notification Access = NO, פתח הגדרות מאזין התראות.
  • Battery optimization = NO: exempt app from optimization.
    אם Battery optimization = NO, החרג את האפליקציה.
  • Write settings = NO: grant from power helper menu.
    אם Write settings = NO, אשר דרך Power helper.
