📺

Android TV

כל מכשיר Android TV (גרסה 5.0 ומעלה)

  • Local TV Apps
  • שאומי Mi Box / Mi TV Stick
  • NVIDIA Shield
  • Google Chromecast with Google TV
  • סטרימרים אחרים עם Android TV

מחשב

Mac או Windows באותה רשת WiFi

  • macOS 10.15+ או Windows 10/11
  • חיבור לאותה רשת WiFi/LAN כמו ה-TV

  • הנתב לא חוסם תקשורת בין מכשירים

7 לפי יצרן

📺 Local TV Apps (ממירים)

Settings → Device Preferences → About → Build

לחץ 7 פעמים על "Build". אחר כך:

Settings → Device Preferences → Developer Options → USB Debugging

💡 IP נמצא ב: Settings → Device Preferences → About → Status

📺 Xiaomi Mi Box / Mi TV Stick

Settings → About → Build

לחץ 7 פעמים. אחר כך:

Settings → Developer Options → USB Debugging

💡 IP נמצא ב: Settings → Network & Internet → [שם הרשת]

📺 NVIDIA Shield

Settings → Device Preferences → About → Build

לחץ 7 פעמים. אחר כך:

Settings → Device Preferences → Developer Options → Network Debugging

⚠️ ב-NVIDIA Shield זה נקרא "Network Debugging" ולא USB Debugging

📺 Google Chromecast / Google TV

Settings → System → About → Android TV OS Build

לחץ 7 פעמים. אחר כך:

Settings → System → Developer Options → USB Debugging

💡 IP: Settings → Network & Internet → [שם הרשת] → Status

8 פתרון בעיות

❌ "Cannot connect" — לא מצליח להתחבר ▼
  • בדוק שה-TV והמחשב על אותה רשת WiFi
  • בדוק שה-ADB Debugging מופעל ב-Developer Options
  • נסה לכבות ולהפעיל מחדש את ה-TV
  • חלק מהנתבים חוסמים תקשורת בין מכשירים — בדוק "AP Isolation" בהגדרות הנתב
  • נסה פורט אחר: adb connect IP:41635 במקום :5555
⚠️ "Unauthorized" — דורש אישור ▼
  • חפש הודעת אישור על מסך ה-TV
  • בחר "Always allow from this computer"
  • אם אין הודעה — כבה ADB Debugging, הפעל מחדש, ונסה שוב
  • Run: adb kill-server && adb start-server
🛡️ Play Protect חוסם את ההתקנה ▼
  • פתח Google Play Store ב-TV
  • לך ל-Settings → Play Protect
  • כבה את "Scan apps with Play Protect"
  • הרץ את ה-Installer שוב
  • אחרי ההתקנה — מומלץ להפעיל חזרה
🔄 שירות הנגישות לא עובד ▼
  • הרץ את ה-Installer שוב — הוא מפעיל את השירות אוטומטית
  • בדוק שהאפליקציה מותקנת: חפש "Stream Keep Alive" ב-Apps
  • פתח את האפליקציה ובדוק אם כתוב "✅ פעיל"
  • בדוק שהגרסה המותקנת היא לפחות 1.7.2 (10)
  • אם לא — לחץ 3 פעמים על "הגדרות נגישות" והפעל ידנית
🍎 macOS חוסם את הסקריפט ▼
  • לחץ ימני על הקובץ → "Open" (במקום לחיצה כפולה)
  • או: System Preferences → Security & Privacy → "Allow Anyway"
  • או הרץ בטרמינל: chmod +x install_mac.command && ./install_mac.command
🪟 Windows SmartScreen חוסם ▼
  • לחץ על "More info"
  • לחץ על "Run anyway"
  • הסקריפט בטוח — הוא רק מריץ ADB commands
📺 ההודעה עדיין מופיעה ▼
  • ודא שהאפליקציה מציגה "✅ פעיל"
  • חכה — השירות מזהה את ההודעה וסוגר אותה תוך 1-2 שניות
  • אם לא עובד — הסגנון של ההודעה ייתכן ושונה מהצפוי. פנה אלינו עם צילום מסך
