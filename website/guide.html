מדריך התקנה

מרכז קישוריות לטלוויזיה
English

1) התקנה

  1. הורד והרץ מתקין מדף הבית.
  2. השתמש בערוץ main אלא אם צריך release.
  3. אמת versionName/versionCode בפלט המתקין.

2) הפעלת הרשאות

  1. נגישות: פעיל
  2. גישת התראות: מומלץ
  3. החרגת סוללה: מומלץ
  4. Write Settings: אופציונלי אך מומלץ

3) הפעלת סשן

לחץ הפעל כלי עזר ובדוק שהצ'קליסט מציג YES.

4) פתרון תקלות

  • אם Notification Access = NO, פתח הגדרות מאזין התראות.
  • אם Battery = NO, החרג את האפליקציה מאופטימיזציה.
  • אם Write Settings = NO, אשר דרך תפריט Power.
