English דף הבית

שלב 1 - התקנה

  1. הורד והרץ מתקין Mac/Windows.
  2. בחר ערוץ main (מומלץ לרוב המשתמשים).
  3. ודא שבפלט המתקין מופיעה הגרסה המותקנת (`versionName/versionCode`).

שלב 2 - הרשאות חיוניות

  1. Accessibility: הפעלה חובה.
  2. Notification Access: מומלץ מאוד.
  3. Battery Optimization Exemption: מומלץ ליציבות.
  4. Write Settings: אופציונלי אך מומלץ להקשחת זמן כיבוי מסך.

שלב 3 - בדיקת מוכנות

פתח את האפליקציה ובדוק שבצ'קליסט רוב השורות מסומנות YES. אם חסרה הרשאה, השתמש בכפתורי Fix או בתפריט Power & System.

פתרון בעיות נפוצות

  • Notification Access = NO → פתח הגדרות מאזין התראות ואשר גישה.
  • Battery Exemption = NO → החרג את האפליקציה מאופטימיזציית סוללה.
  • Write Settings = NO → אשר הרשאה דרך Power & System.
  • Timeout Override = NO → אשר Write Settings והפעל מחדש את כלי העזר.
