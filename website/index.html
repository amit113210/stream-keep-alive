Hotspot / connectivity helper

כניסות מהירות לזרימות רשת מקומיות והגדרות נקודת גישה.

Quick settings shortcuts

Quick setup | התקנה מהירה

  1. Install via Mac/Windows installer (channel main recommended).
    התקן דרך מתקין Mac/Windows (ערוץ main מומלץ).
  2. Enable Accessibility for TV Connectivity Hub.
    הפעל נגישות עבור TV Connectivity Hub.
  3. Enable Notification Access (recommended).
    הפעל גישת התראות (מומלץ).
  4. Open Power & System helper and apply recommendations.
    פתח תפריט Power & System ויישם המלצות.
  5. Start Utility and verify readiness status.
    הפעל Utility ובדוק סטטוס מוכנות.

FAQ

No, behavior can vary by OEM and firmware.

No, normal Android TV behavior is preserved.
