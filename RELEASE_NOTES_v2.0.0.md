# TV Connectivity Hub v2.0.0

## Major Product Pivot + Rebrand

This release pivots the project from a streaming interruption helper into a TV connectivity utility.

### Highlights
- Rebrand to **TV Connectivity Hub** across app, website, installer, and documentation.
- New utility-first product messaging focused on:
  - hotspot/connectivity helper flows
  - quick system settings access
  - readiness dashboard and diagnostics
  - setup and troubleshooting guidance
- Updated app UI copy and visual styling for the new utility identity.
- Updated installer copy (Mac/Windows) to remove outdated anti-streaming claims.
- Rewritten website homepage and setup guide for the new product scope.
- Rewritten README and added focused docs:
  - `QUICK_START.md`
  - `FEATURES.md`
  - `PERMISSIONS_AND_SETTINGS.md`
  - `TROUBLESHOOTING.md`
