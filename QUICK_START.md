# Quick Start - TV Connectivity Hub

1. Install the APK using the Mac or Windows installer.
2. Open **TV Connectivity Hub** on your Android TV.
3. Enable Accessibility when prompted.
4. Enable Notification Access (recommended).
5. Open **Power & System Settings** and apply recommended hardening:
   - Battery optimization exemption
   - Screen timeout permission (WRITE_SETTINGS)
6. Press **Start Utility**.
7. Validate the checklist reads YES for key items.

If something is missing, use **More Actions** and **Power & System Settings** to fix it directly.
