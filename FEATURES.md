# Features - TV Connectivity Hub

## 1. Connectivity Helper
- Quick entry points for Wi-Fi/network/hotspot-related workflows.
- Device behavior varies by Android TV OEM and firmware.

## 2. Utility Session
- User-initiated foreground companion session.
- Keeps session state visible via persistent notification.

## 3. Readiness Dashboard
- Accessibility enabled
- Notification access enabled
- Battery optimization exempt
- Write settings granted
- Screen-timeout hardening status

## 4. Power & System Shortcuts
- Battery optimization settings
- Write settings permission screen
- App info
- Accessibility settings
- Notification listener settings

## 5. Telemetry Panel
- Session/runtime state
- Playback signal availability
- Gesture and dialog diagnostics
- Power hardening apply/restore state
